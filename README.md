init-rn-env
===

Initialize react-native development environment

安装与更新
-------

    $ [sudo] npm install init-rn-env -g

升级 `init-rn-env` 请用如下命令。

    $ [sudo] npm update init-rn-env -g
    

使用
------

Show default info

    $ ire  

Initialize react-native development environment in dirPath

    $ ire dirPath
